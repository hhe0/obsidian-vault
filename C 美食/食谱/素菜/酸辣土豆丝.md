#### 一、食材准备
##### 食材组

| 食材  | 用量   |
| --- | ---- |
| 土豆  | 1-2个 |
| 青椒  | 1个   |

##### 辅料组

| 辅料  | 用量  |
| --- | --- |
| 蒜   | 适量  |
| 生姜  | 适量  |


##### 香料组

| 香料  | 用量  |
| --- | --- |
| 干辣椒 | 适量  |
| 花椒  | 十几粒 |

##### 酱料组

| 酱料  | 用量  |
| --- | --- |
| 白醋  | 适量  |
| 盐   | 适量  |

#### 二、烹饪步骤
* 土豆洗净用刮皮刀刮去外皮切成厚薄均匀的丝，青辣椒切丝，蒜切碎；
* 将土豆丝放入凉水中多清洗几次，将淀粉都洗掉后沥干水分；
* 锅中倒油，放入花椒小火炸香，把花椒捞出不要；
* 放入葱、姜、干辣椒爆香，倒入土豆丝翻炒均匀后，立刻加入白醋继续翻炒约1分钟；
* 加入青辣椒、蒜、盐，翻炒均匀后关火即可。
