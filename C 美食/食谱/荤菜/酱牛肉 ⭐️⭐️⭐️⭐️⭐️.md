#### 一、食材准备
##### 食材组

| 食材  | 用量  |
| --- | --- |
| 牛腱子 | 1公斤 |

##### 辅料组

| 辅料  | 用量       |
| --- | -------- |
| 洋葱  | 半个，40g左右 |
| 生姜  | 30g      |

##### 香料组

| 香料  | 用量  |
| --- | --- |
| 花椒  | 十几粒 |
| 八角  | 2个  |
| 桂皮  | 1个  |
| 香叶  | 2片  |
| 干辣椒 | 适量  |

##### 酱料组[[
]][[[[
]]]]

| 酱料  | 用量   |
| --- | ---- |
| 生抽  | 200g |
| 甜面酱 | 120g |
| 豆腐乳 | 20g  |

#### 二、烹饪步骤
##### 去腥
* 将牛肉反复清洗，泡水3个小时左右。
##### 第一酱
* 将牛肉表面的水分控干，加入生抽，放入冰箱隔夜浸泡。
##### 焯水
* 将牛肉连通生抽一起冷水下锅，进行焯水。
##### 第二酱
* 加入香料、洋葱、生姜和甜面酱。
##### 第三酱
* 加入豆腐乳。
##### 烹煮
* 小伙烹煮2小时15分钟，然后尝咸淡，补适量的盐。
